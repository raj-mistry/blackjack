# BlackJack Android app
A Software Engineering project to create a black-jack simulator completed with a fully functional virtual gambling system backed up by cloud storage through Google's platform. Currently unavailable through the playstore, but we can send you the apk if you'd like :D

This project was created as a software engineering final project.
Software used: Android Studio
Languages: Java
API's: 
Google Firestore Firebase Real-Time Database and Authentication.
JUnit testing
Espresso Testing

This software is a black-jack simulator with a betting system.
The database stores user information such as games lost and won, win %, player bank account balance.
Prior to entering the game, the user is prompted to place a bet.
The game features a graphical user interface of Buttons and images of the player's hand.
The player plays against an AI dealer.

Player games can be seen in a recycler view.
